понравился PerfectCorp, вот другие готовые решения именно под одежду (уровень API/SDK и боевой прод). Коротко: что это, чем полезно, и где почитать.

Enterprise-класс (API из коробки)

Google Vertex AI — Virtual Try-On API: чистый REST/SDK, входы personImage + productImages, детальные параметры (кол-во сэмплов, сохранение в GCS). Хорошо для C#/Node бекенда и масштабирования. 


AWS Nova Canvas (Virtual Try-On): новый модуль VTO, принимает source (человек) + reference (гармент), есть режимы «stitching» (BALANCED/SEAMLESS/DETAILED) и тонкие настройки — удобно, если уже живёте в AWS. 


Специализированные fashion-платформы (SaaS + интеграции)

Veesual: VTO/«mix&match» для моды, заточено под e-com витрины и конверсию; фокус на визуальный мерчандайзинг и персонализацию. Есть интеграции и кастомизация под бренд. 


Revery AI: позиционируется как «виртуальная примерочная через API»; ориентир на быстрое встраивание для ритейлеров. (Молодая, но именно API-first). 


WEARFITS: VTO + size-fitting (3D/AR), есть публичная дока API (GitHub). Хорошо, если помимо «примерить на фото» хочешь «подбор размера». 


Style.me: «виртуальная примерочная» с подбором размера и 3D-визуализацией; больше SDK/виджет, чем «сырой» генеративный API, но боевой для одежды. 


Kivisense: WebAR/VTO для одежды и аксессуаров; сильна в realtime-треккинге на вебе. Подойдёт, если нужна «живость» и AR-интерактив, а не только рендер по фото. 


Zyler: именно «одежда по фото»; есть гайды по интеграции (в т.ч. Direct Integration). Часто ставят крупные ретейлеры. 

Reactive Reality (PICTOFiT): мощный SDK/веб-SDK для VTO одежды (2D/3D), плюс контент-сервис (аватары/ассеты). Выше порог входа, но гибкость «под магазин». 


Новые/альтернативные

VModel / vmodel.ai Try-On API: лёгкий REST для «переодевания» по фото (категории/маски). Подходит для быстрых экспериментов. 


Как выбрать быстро (TL;DR)

Нужно «как у PerfectCorp», но с большим контролем и SDK-ами → Google Vertex AI VTO или AWS Nova Canvas (если ваш стек на AWS). 


Нужен готовый fashion-виджет/вид с минимальным кодом → Style.me / Veesual / Zyler. 


Нужен VTO + подбор размера/аватары → WEARFITS или Reactive Reality (PICTOFiT). 


Хочешь быстрое PoC-API → VModel (или Revery, если дадут доступ к API).
